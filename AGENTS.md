# AGENTS.md

## Purpose

This repository publishes reusable Codex skills.

## Repository Layout

- `skills/` contains the source for distributable skills.
- `docs/` contains supporting documentation when needed.

## Working Rules

- Keep changes focused on published skill sources and supporting docs.
- Do not commit generated artifacts that are not part of the published source.

## Release Rule

- Prefer updating the source directory under `skills/` instead of committing generated local artifacts.
