# skill-creator

This repository publishes reusable Codex skills, starting with `skills/general-project`.

## Included Skill

- `general-project`: bootstrap a repository with a reusable `.codex` workflow kit, then execute general project work with strict scope and proportionate verification.

## Repository Structure

- `skills/`: published skill source
- `docs/`: supporting notes and documentation

## Updating The Skill

Edit the source under `skills/general-project/`. Do not rely on local zip packages as the source of truth.
